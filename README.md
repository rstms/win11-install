# win11 installer

customized win11 installer USB builder 

    - bypass hardware dependencies
    - bypass forced microsoft account
    - bypass bloat / telemetry / AI

## requirements 
    win11 ISO
    USB drive
    product key
    env var HOSTNAME

## autounattend.xml 

generated by https://schneegans.de/windows/unattend-generator
 - load web editor with template.xml
 - adjust settings using web interface
 - enter __MACRO_NAME__ for values to be expanded by generate-xml script
 - save results back to template.xml

## generate-xml

expands __MACRO_VARIABLES__ in template.xml using values in secrets/${HOSTNAME}/XML_*
each file in the secrets dir XML_MACRO_NAME contains the value for __MACRO_NAME__

## configuration

These files must exist in directory secrets/$HOSTNAME/

 - XML_PRODUCT_KEY
 - XML_HOSTNAME
 - XML_IMAGE
 - XML_USERNAME
 - XML_DISPLAY_NAME
 - XML_PASSWORD
 - XML_ADMINISTRATOR_PASSWORD
